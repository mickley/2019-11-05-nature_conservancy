Programming & Data Analysis with R

  • General R Help
    • rdrr.io: Nicely formatted help for functions in R packages. Easier to read and even lets you run examples in your browser!
    • RStudio Cheatsheets: Especially recommend data wrangling, data visualization, and R markdown.
    • stackoverflow.com: Great place to ask for or search for help with specific problems. Many package authors monitor this website.
    • Tidyverse (dplyr, tidyr, ggplot) Style Guide: Write neat readable code
    • R Cookbook: Online textbook of sorts
  • Making ggplot Graphs
    • ggplot2 Online Documentation: Shows all the different geometries/plot types you can add, with examples
    • R Cookbook: Section on graphing
    • ggplot2 Essentials: This website has a wealth of information on more advanced or complicated graphs
    • R Graph Catalog: Lots of graph examples with code by another Software Carpentry instructor
    • Cowplot package: A theme and some extra tools (combining graphs) for making publication-ready graphs in ggplot2
  • Data Wrangling
    • dplyr package documentation: Man pages and vignettes showing how to use additional dplyr functions
    • tidyr package documentation: Man pages and vignettes showing how to use additional tidyr functions
    • Data Transformation Cheat Sheet: RStudio cheatsheet on data manipulation
    • R Cookbook: Section on data manipulation
    • RStudio webinar
  • R Markdown & R Notebooks
    • R Markdown Official Website: Lots of documentation, examples, and webcasts here
    • Quick tour webcast
    • Additional lessons with R Markdown
  • Workshop Code
    • Day 1: Intro R
    • Day 1: programming with R and ggplot
    • Day 2: Data wrangling in R
